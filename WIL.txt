이번주는 시간이 엄청 빠르게 지나가는 한 주였다.

처음 항해99를 시작하고 바로 미니프로젝트를 시작하고,

팀원들과 회의를 통해 미니프로젝트 주제와 방향성 개발사항등을 결정했다.

우리가 결정한 것은 끌이라는 이름의 홈페이지 제작이였고

끌은 서퍼들에게 해변의 파고, 기온, 수온 등을 서핑에 필요한 정보들을 제공하는

홈페이지이다.

처음에는 API를 통해 실시간 영상을 제공하려고 했으나 API 구하기가 어려워서 포기 하고,

기상청과 해양수산부 자료, 지도자료등을 통해 API를 가져와 파고, 기온, 수온등을 제공하였다.

또한 유저들에게 다양한 서비스를 위해 JWT를 이용한 로그인을 구현하여 서로 정보를 교류 할 수 있게 만들었다.

이번 프로젝트를 하면서 제일 어려웠던건 API 구하는거였다.

공공기관에서 제공하는 API는 한계가 있어 구현하기가 어려웠고, 제공하는 자료들을 다시 한번 취합하는게 쉽지 않았다.

그래도 팀원들과 협업을 통해 여러 아이디어를 조합해 구현했고 이를 통해 한단계 성장한것 같다.
